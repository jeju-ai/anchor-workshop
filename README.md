## 참여 방법

1. **사전학습**: `workshop/pre-learning/` 폴더 순서대로 읽고 환경 설정 완료
2. **체크리스트**: `workshop/pre-learning/checklist.md` 항목 모두 완료 후 Slack 인증
3. **세션 참여**: 날짜에 맞춰 오프라인 세션 참석
4. **경진대회**: 개인 또는 팀(2~3인) 구성 후 `challenge/templates/submission-readme.md` 참고하여 제출

## 저장소 구조

```
anchor-workshop/
├── workshop/
│   ├── pre-learning/   사전학습 자료 (4개)
│   ├── sessions/       세션별 자료 (6개)
│   └── exercises/      실습 파일 (예산 데이터, 보고서 초안, 회의록, 공문 양식)
├── challenge/
│   ├── rules/          경진대회 규정 (카테고리, 심사기준, 시상)
│   ├── templates/      제출 양식
│   ├── submissions/    팀 제출물 (PR로 추가)
│   └── results/        결과 발표
├── prompt-library/     업무별 프롬프트 모음 (기여 환영)
└── guidelines/         AI 사용 정책, Slack 문화 가이드
```
